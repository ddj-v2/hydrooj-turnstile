# hydrooj-turnstile

This plugin adds Cloudflare Turnstile verification to the Hydro OJ platform. It allows administrators to configure Turnstile for various pages, enhancing security by preventing automated submissions.

這個插件為 Hydro OJ 平台添加了 Cloudflare Turnstile 驗證。它允許管理員為各種頁面配置 Turnstile，通過防止自動提交來增強安全性。
